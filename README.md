# Portfólio Gabriel Andrade — Repositório reorganizado

Este projeto foi separado em arquivos independentes para facilitar manutenção, edição e evolução.

## Estrutura

```txt
portfolio-gabriel-reorganizado/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    ├── img/
    │   └── README.md
    └── docs/
```

## Onde editar cada parte

- `index.html`: estrutura da página, textos, links, seções, cards e conteúdo.
- `css/style.css`: aparência visual, cores, espaçamentos, responsividade e animações.
- `js/main.js`: cursor personalizado, efeito de digitação, animações de scroll e comportamento da navbar.
- `assets/img/`: imagens do projeto, como sua foto de perfil.
- `assets/docs/`: documentos para download, como currículo em PDF, certificados etc.

## Seções organizadas no CSS

O `style.css` está dividido por blocos comentados:

1. Reset e variáveis globais
2. Base do projeto
3. Cursor personalizado
4. Navegação
5. Hero / apresentação inicial
6. Seções globais
7. Sobre mim
8. Habilidades
9. Projetos
10. Cursos e certificações
11. Contato
12. Rodapé
13. Animações
14. Scrollbar
15. Responsividade

## Seções organizadas no JavaScript

O `main.js` também está dividido em blocos comentados:

1. Cursor personalizado
2. Efeito de digitação no Hero
3. Animação Reveal ao rolar a página
4. Alteração visual da navbar no scroll

## Como usar

Abra o arquivo `index.html` no navegador. Para publicar, envie a pasta inteira para Netlify, GitHub Pages ou outro serviço de hospedagem estática.
